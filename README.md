# Smart Timecard (勤怠管理アプリ)

TypeScript + React + Vite で作成された、シンプルで高機能な勤怠管理アプリケーションです。

## 主な機能

- **柔軟な打刻システム**: 1日に何度でも出勤・退勤が可能（再出勤対応）。
- **休憩管理**: 勤務時間中の休憩開始・終了を個別に記録。
- **個人ダッシュボード**: 今月の合計稼働時間や概算給与をリアルタイムに確認。
- **管理者モード**: 
  - 全作業員の稼働状況をグラフと数値で可視化。
  - 月次詳細データの確認およびCSVエクスポート。
  - ユーザー管理（PIN設定含む）。
- **レスポンシブデザイン**: PC、タブレット、スマートフォンに対応。

## 技術スタック

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand (状態管理)
- Lucide React (アイコン)
- date-fns (日付操作)

## 開発環境の実行

```bash
npm install
npm run dev
```
